# Fonts

The UI uses **Graphik** (KSE's brand typeface). Graphik is a commercial font by
Commercial Type and is **not** committed to this repo for licensing reasons.

To restore the exact KSE look, place these files here:

- `Graphik-Regular-Web.woff2`
- `Graphik-Regular-Web.woff`
- `Graphik-Medium-Web.woff2`
- `Graphik-Medium-Web.woff`

Without them the app falls back to Inter / system-ui (declared in `globals.css`).
